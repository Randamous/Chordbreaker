# TODO / Roadmap

Этот файл — активный список следующих работ. Закрытые исторические задачи из старого TODO удалены из активного списка: источник истины по уже реализованному поведению — тематические документы `docs/02_*`–`docs/07_*` и `docs/08_recent_changes.md`.

## Текущее направление проекта

Проект уже имеет рабочий core-loop: выбор персонажа, забег по комнатам, награды, магазин, босса, переходы между этажами, сохранение run-checkpoint и account-level meta-profile с валютой/known-items. Следующий большой слой — превратить забег в полноценный roguelite с городом/хабом, коллекцией предметов, выбором биома/сложности и более разнообразным контентом.
